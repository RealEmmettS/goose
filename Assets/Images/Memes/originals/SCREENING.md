# Original Meme Screening

M9 copies screened original meme assets for personal-use builds only when they do not carry old
developer references, donation/Patreon links, social handles, or old-project branding.

`Meme2.png` from the reference app is intentionally not bundled in this directory because visual
screening found a baked-in social handle watermark. The in-house custom counterpart remains under
`Assets/Images/Memes/custom/`.
