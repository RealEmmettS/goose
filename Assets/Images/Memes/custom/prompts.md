# Custom M9 Meme Prompts

These are complete custom in-house counterparts, not variations of the original images. The final
PNG assets were regenerated with the built-in `image_gen` tool on 2026-06-27, using one distinct
scene prompt per counterpart plus the shared base style prompt below. The workspace copies are
downscaled to a maximum 960px edge for practical collect-window sizing; the generated source PNGs
remain under the Codex image output directory.

Base style prompt:

> Draw {MEME_IMAGE_PROMPT} in the most clumsy, scribbly, and utterly pathetic way possible. Use a white background, and make it look like it was drawn in an old computer painting program with a mouse. It should be vaguely similar but also not really, kind of matching but also off in a confusing, awkward way, with that low-quality pixel-by-pixel feel that really emphasizes how ridiculously bad it is. Actually, you know what, whatever, just draw it however you want.
